## 3-2 연산자의 종류
연산자는 피연산자의 수에 따라 단항, 이항, 삼항 연산자로 구분

부호 연산자, 증감 연산자 - 단항 연산자
조건 연산자 - 삼항 연산자
그 외의 연산자 - 이항 연산자

![alt text](image.png)

#### 단항 연산자
부호 연산자(+,-), 증감 연산자(++,--), 논리 부정 연산자(!)

##### 부호 연산자(+,-)

| 연산식 |  | 설명 |
|-------|-------|-------|
| + | 피연산자 | 피연산자의 부호 유지 |
| - | 피연산자 | 피연산자의 부호 변경 |
- boolean타입, char타입을 제외한 나머지 기본 타입에 사용
- +,-는 산술 연산자와 부호 연산자 둘 다 사용 가능

부호 연산자일 경우
정수 및 실수 리터럴 앞에 붙여 양수 및 음수를 표현
```java
int i1 = +100;
  ```
- +연산자는 변수값 부호를 유지
-연산자는 양수는 음수, 음수는 양수로 바꿈
- byte 타입 값을 부호 연산하면 int 타입 값으로 바뀜

#### 증감 연산자(++,--)

##### 증감 연산자 (++)
- 변수의 값을 **1 증가**시킴
- 전위형(++x): 값을 먼저 증가시킨 후 사용
➡️ 흐름: 변수 → +1 → 사용

- 후위형(x++): 값을 사용한 후 증가시킴
➡️ 흐름: 변수 → 사용 → +1

##### 감소 연산자 (--)
- 변수의 값을 **1 감소**시킴
- 전위형(--x): 값을 먼저 감소시킨 후 사용
➡️ 흐름: 변수 → -1 → 사용

- 후위형(x--): 값을 사용한 후 감소시킴
➡️ 흐름: 변수 → 사용 → -1

[증감 연산자](_0302_2.java)

#### 단항 연산자
- 단항 연산자는 피연산자가 **하나만 필요한 연산자**이다.
- 값 자체를 직접 변경하거나, 조건을 판별하거나, 단순히 값을 증감시킬 때 사용된다.
- 대표적으로 **부호 연산자(+,-)**, **증감 연산자(++/--)**, **논리 부정 연산자(!)** 가 있다.

---

##### 논리 부정 연산자 (!)
- 불리언 타입(boolean)에만 적용할 수 있다.
- true는 false로, false는 true로 **논리값을 반전**한다.
- 조건식에서 자주 사용되며, 특정 조건을 **부정**할 때 유용하다.

예시:
```java
boolean isActive = true;
System.out.println(!isActive); // false

int score = 80;
System.out.println(!(score > 90)); // true (90점 이상이 아님)
```

- 주의:
  - 숫자나 문자열에는 사용할 수 없다.
  - 반드시 boolean 결과값에만 적용 가능하다.
- 활용:
  - 로그인 상태가 아닐 때: `if (!isLoggedIn)`
  - 데이터가 비어있지 않을 때: `if (!list.isEmpty())`

---

##### 부호 연산자 (+, -)
- 숫자 타입(정수, 실수) 앞에 붙여 사용하는 연산자.
- `+` : 값을 그대로 유지한다. (사실상 의미는 거의 없음)
- `-` : 값의 부호를 반대로 바꾼다.

예시:
```java
int x = 10;
int y = -x;  // -10
int z = -y;  // 10
```

- 주의:
  - 변수 자체의 값이 바뀌는 것이 아니라 **연산 결과**만 달라진다.
- 활용:
  - 양수/음수 전환
  - 수학 수식에서 부호 처리

---

##### 증감 연산자 (++, --)
- 변수의 값을 **1씩 증가** 또는 **1씩 감소**시키는 연산자.
- 위치에 따라 의미가 달라진다.
  - **전위형(++a)** : 연산 먼저, 그 다음 식에 반영.
  - **후위형(a++)** : 현재 값 먼저 사용, 그 다음 연산.

예시:
```java
int a = 5;
System.out.println(++a); // 6 (전위, 먼저 증가)
System.out.println(a++); // 6 (후위, 출력 후 증가)
System.out.println(a);   // 7
```

- 활용:
  - 반복문에서 루프 카운터를 증가/감소시킬 때 자주 사용된다.
- 주의:
  - 전위형과 후위형의 차이를 반드시 이해해야 한다.

---

#### 이항 연산자
- 두 개의 피연산자를 사용하여 연산을 수행하는 연산자이다.
- 크게 **산술 연산자**, **문자열 결합 연산자**, **비교 연산자**, **논리 연산자**, **대입 연산자** 등이 있다.

---

##### 산술 연산자 (+, -, *, /, %)
- 기본적인 수학 계산을 수행한다.
- `+` : 덧셈  
- `-` : 뺄셈  
- `*` : 곱셈  
- `/` : 나눗셈 (몫)  
- `%` : 나머지  

예시:
```java
int a = 10, b = 3;
System.out.println(a + b); // 13
System.out.println(a - b); // 7
System.out.println(a * b); // 30
System.out.println(a / b); // 3
System.out.println(a % b); // 1
```

- 주의:
  - `/` 연산에서 정수끼리 나누면 **몫만** 남는다.
  - 실수 결과를 원하면 피연산자 중 하나를 double이나 float로 변환해야 한다.
```java
System.out.println(10 / 4);      // 2
System.out.println(10 / 4.0);    // 2.5
```

---

##### 문자열 결합 연산자 (+)
- 문자열을 서로 연결하거나, 문자열과 다른 타입을 결합할 때 사용된다.
- 숫자 + 문자열 → 문자열 결합으로 처리된다.

예시:
```java
String s1 = "Hello";
String s2 = "World";
System.out.println(s1 + " " + s2); // Hello World

int age = 20;
System.out.println("나이: " + age); // 나이: 20
```

- 주의:
  - 숫자 + 숫자는 덧셈 연산, 문자열이 하나라도 포함되면 문자열 결합이 된다.
```java
System.out.println(1 + 2 + "3"); // "33"
System.out.println("1" + 2 + 3); // "123"
```

---

##### 비교 연산자 (<, <=, >, >=, ==, !=)
- 두 값을 비교하고 결과를 boolean으로 반환한다.
- `<, <=, >, >=` : 대소 비교  
- `==` : 값이 같은지 비교  
- `!=` : 값이 다른지 비교  

예시:
```java
int x = 5, y = 10;
System.out.println(x < y);  // true
System.out.println(x >= y); // false
System.out.println(x == y); // false
System.out.println(x != y); // true
```

- 주의:
  - 문자열 비교 시 `==`는 참조 비교이므로 `.equals()`를 사용해야 한다.
```java
String a = new String("hello");
String b = new String("hello");
System.out.println(a == b);       // false
System.out.println(a.equals(b));  // true
```

---

##### 논리 연산자 (&&, ||, ^, !)
- 논리식을 결합하거나, 조건을 더 복잡하게 표현할 때 사용된다.
- `&&` : 논리곱 (AND) → 둘 다 true일 때만 true  
- `||` : 논리합 (OR) → 하나라도 true면 true  
- `^` : 베타적 논리합 (XOR) → 서로 다를 때 true  
- `!` : 논리 부정 (NOT) → true ↔ false  

예시:
```java
boolean a = true, b = false;
System.out.println(a && b); // false
System.out.println(a || b); // true
System.out.println(a ^ b);  // true
System.out.println(!a);     // false
```

- 주의:
  - `&&` 와 `||`는 **단축 평가(short-circuit)**를 수행한다.
  - 왼쪽 결과로 전체 결과가 결정되면 오른쪽은 평가하지 않는다.

---

##### 대입 연산자 (=, +=, -=, *=, /=, %=)
- 값을 변수에 저장할 때 사용한다.
- `=` : 기본 대입 연산자  
- `+=, -=, *=, /=, %=` : 산술 연산 후 그 결과를 다시 변수에 저장한다.

예시:
```java
int num = 10;
num += 5; // num = num + 5 → 15
num -= 3; // num = num - 3 → 12
num *= 2; // num = num * 2 → 24
num /= 4; // num = num / 4 → 6
num %= 5; // num = num % 5 → 1
System.out.println(num);
```

---

#### 삼항 연산자
- 삼항 연산자는 조건식의 결과에 따라 두 가지 값 중 하나를 선택한다.
- 형식: `조건식 ? 값1 : 값2`
- 조건식이 true면 `값1`, false면 `값2`가 선택된다.

예시:
```java
int score = 85;
String result = (score >= 60) ? "합격" : "불합격";
System.out.println(result); // 합격
```

- 특징:
  - if-else 문을 간단히 표현할 수 있다.
  - 가독성이 좋아지고 코드가 간결해진다.

---

### 정리
1. **단항 연산자**
   - `!` : 논리 부정  
   - `+,-` : 부호 연산자  
   - `++,--` : 증감 연산자  

2. **이항 연산자**
   - 산술 연산자: `+,-,*,/, %`  
   - 문자열 결합 연산자: `+`  
   - 비교 연산자: `<, <=, >, >=, ==, !=`  
   - 논리 연산자: `&&, ||, ^, !`  
   - 대입 연산자: `=, +=, -=, *=, /=, %=`  

3. **삼항 연산자**
   - `조건식 ? 값1 : 값2`  
   - 조건에 따라 두 가지 결과 중 하나 선택

# 마무리

## ▶ 5가지 키워드로 끝내는 핵심 포인트

• **증감 연산자**: ++, --를 말하며 변수의 값을 1씩 증가, 1씩 감소시킵니다.

• **비교 연산자**: ==, != 등을 말하며 값이 같은지, 다른지를 비교하고 boolean 값을 산출합니다.

• **논리 연산자**: &&, !!, ! 등을 말하며 논리곱, 논리합, 논리 부정을 수행하고 boolean 값을 산출합니다.

• **대입 연산자**: =, +=, -= 등을 말하며 오른쪽의 값을 왼쪽에 대입하거나 연산 후 대입합니다.

• **삼항 연산자**: (조건식) ? A : B를 말하며 조건이 true이면 A를 산출하고, false이면 B를 산출합니다.

```java
byte b = 5;
b = -b;
int result = 10 / b;

```

**풀이**: 
- 컴파일 에러는 `b = -b;` 라인에서 발생합니다.
- 이유: `-b`는 int 타입으로 승격되어 연산되지만, byte 변수 b에 int 값을 대입할 수 없기 때문입니다.
- 해결방법: `b = (byte)(-b);` 또는 `b *= -1;`로 수정해야 합니다.

### 4. 534자루의 연필을 30명의 학생들에게 똑같은 개수로 나누어 줄 때 1인당 몇 개를 가질 수 있고, 몇 개가 남는지를 구하는 코드입니다. ①~②에 들어갈 알맞은 코드를 작성하세요.

```java
int pencils = 534;
int students = 30;

//학생 1명이 가지는 연필 개수
int pencilsPerStudent = (         ①         );
System.out.println(pencilsPerStudent);

//남은 연필 개수
int pencilsLeft = (         ②         );
System.out.println(pencilsLeft);
```

**풀이**:
```java
int pencils = 534;
int students = 30;

//학생 1명이 가지는 연필 개수
int pencilsPerStudent = (pencils / students);  // ① 답: pencils / students
System.out.println(pencilsPerStudent);

//남은 연필 개수
int pencilsLeft = (pencils % students);        // ② 답: pencils % students
System.out.println(pencilsLeft);
```
- ① `pencils / students = 534 / 30 = 17` (1인당 17개)
- ② `pencils % students = 534 % 30 = 24` (24개 남음)

### 5. 다음 코드를 실행하면 출력 결과로 5를 기대했는데 4가 출력되었습니다. 어디에서 잘못 작성된 것인가요?

```java
int var1 = 5;
int var2 = 2;
double var3 = var1 / var2;
int var4 = (int) (var3 * var2);
System.out.println(var4);
```

**풀이**:
- 문제점: `var1 / var2`에서 정수 나눗셈이 수행되어 `5 / 2 = 2`가 됩니다.
- `var3 = 2.0`, `var4 = (int)(2.0 * 2) = 4`
- **해결방법**: 
```java
double var3 = (double)var1 / var2;  // 또는 var1 / (double)var2
```
- 이렇게 하면 `var3 = 2.5`, `var4 = (int)(2.5 * 2) = 5`

### 7. 다음 코드를 실행하면 출력 결과로 "10%입니다."를 기대했는데 "10%가 아닙니다."가 출력되었습니다. 어디에서 잘못 작성된 것인가요?

```java
float var1 = 10f;
float var2 = var1 / 100;
if(var2 == 0.1) {
    System.out.println("10%입니다.");
} else {
    System.out.println("10%가 아닙니다.");
}
```

**풀이**:
- 문제점: 부동소수점 연산의 정밀도 문제로 `var2`가 정확히 `0.1`이 되지 않습니다.
- **해결방법**:
```java
if(Math.abs(var2 - 0.1) < 0.00001) {  // 오차 범위 내에서 비교
    System.out.println("10%입니다.");
}
```
또는
```java
if(var2 >= 0.09 && var2 <= 0.11) {    // 범위로 비교
    System.out.println("10%입니다.");
}
```

### 9. 키보드로 두 실수를 다음과 같이 입력받습니다(Scanner 이용). 입력받은 첫 번째 수에 두 번째 수를 나는 결과를 "결과값"으로 출력하되, 두 번째 수에 0 또는 0.0이 입력되었을 경우 "결과: 무한대"가 출력되도록 코드를 작성해보세요(Scanner의 next() 사용).

```
첫 번째 수: 7.3 Enter
두 번째 수: 2.5 Enter
-----------------------
결과: 2.92 (or 결과: 무한대)
```

**풀이**:

### 12. 다음 코드는 비교 연산자와 논리 연산자의 복합 연산식입니다. 연산식의 출력 결과를 괄호 ( ) 속에 넣으세요.

```java
int x = 10;
int y = 5;
System.out.println( (x>7) && (y<=5) );    --------------- (        )
System.out.println( (x%3 == 2) || (y%2 != 1) ); --------------- (        )
```

**풀이**:
```java
int x = 10;
int y = 5;
System.out.println( (x>7) && (y<=5) );           // (true)
System.out.println( (x%3 == 2) || (y%2 != 1) );  // (false)
```

**계산 과정**:
- 첫 번째: `(10>7) && (5<=5)` = `true && true` = `true`
- 두 번째: `(10%3 == 2) || (5%2 != 1)` = `(1 == 2) || (1 != 1)` = `false || false` = `false`

## 추가 참고사항

### 연산의 방향과 연산 우선순위
1. **산술 연산자**(+, -, *, /)는 연산 방향이 왼쪽에서 오른쪽으로 수행된다. ✓
2. **대입 연산자**(=, += …)는 연산 방향이 오른쪽에서 왼쪽으로 수행된다. ✓
3. **우선순위와 관계없이** 먼저 연산을 하고 싶다면 괄호 `()`로 묶어준다. ✓
4. **논리 연산자**(&&, ||)는 산술 연산보다 우선 수행된다. ✗

> **올바른 우선순위**: 산술 연산 → 비교 연산 → 논리 연산 순서입니다.
